rpm:unpack

Full name:

org.codehaus.mojo:rpm-maven-plugin:2.2.0:unpack

Description:

Unpack a RPM file. Experimental only. Settings may change in next version

Attributes:

  • Executes as an aggregator plugin.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: generate-resources.

Required Parameters

Name Type Since Description
rpmFile File 2.2.0 RPM file to unpack
User property is: rpm.file.

Optional Parameters

Name Type Since Description
unpackDirectory File 2.2.0 Directory to unpack to
Default value is: ${project.build.directory}/rpm/unpack.
User property is: rpm.unpackDirectory.

Parameter Details

rpmFile:

RPM file to unpack
  • Type: java.io.File
  • Since: 2.2.0
  • Required: Yes
  • User Property: rpm.file

unpackDirectory:

Directory to unpack to
  • Type: java.io.File
  • Since: 2.2.0
  • Required: No
  • User Property: rpm.unpackDirectory
  • Default: ${project.build.directory}/rpm/unpack
